# 絵コンテルール

## 基本書式

- テンプレートは `common/絵コンテテンプレート.png` を基準にする。
- 1枚につき最大8カットを記載できる。
- 画像サイズと縦横比はテンプレートに準拠し、縦長1ページとして扱う。
- ページ上部には「作品名」「話数」「シーン番号」「全体の長さ（秒数）」を入れる。
- SCENE VISUAL（シーンビジュアル）：当該シーンで芝居を開始するキャラクターの立ち位置や、舞台美術を描きこんだメインビジュアルを大きく入れる。ここには、カットを通してキャラクターがどこからどこまで移動するかなどの動きの指示を、矢印などで書き込んでもよい。
- SCENE SUMMARY（シーンサマリー）：当該シーンで描かれる要素を4行程度でまとめる。
- CUT LIST（カットリスト）：最大8個のカットを記載する。カットのタイミングはシーン全体の長さ（秒数）に合わせて記載する。また、各カットは以下の5つの領域で構成する。
  1. カット番号：1から8までの数字を入れる。
  2. カメラ：画面内の動き、カメラワークをを簡潔に書く。`WIDE`、`HANDHELD風`、`POV`、`REVEAL`、`CLOSE` など、撮影意図が分かる語を含める。
  3. CUT IMAGE（カットイメージ）：カットのラフスケッチ
  4. CUT SUMMARY（カットサマリー）：当該カットで描かれる要素を3行程度でまとめる。

## 生成プロンプトテンプレート

GPT Image2.0で絵コンテ画像を生成するためのプロンプトテンプレート。  
後半部分の　[WORK TITLE]～[CUT PLAN]にかけては、描きたいシーンに応じて動的に変える必要がある。  
キャラクターの参照イメージがある場合は[CHARACTERS]に参照画像の読み込み先を記載する。  
舞台背景がある場合は[LOCATION / BACKGROUND]に参照画像を記載する。  
おおまかなコンセプトアートがある場合は[SCENE CONTENT]に参照画像を記載する。  

```
You are filling in a storyboard worksheet for AI video generation.

Use the provided worksheet image as the exact layout template.
Keep the worksheet structure unchanged.
Do not redesign the sheet.
Do not add extra fields.
Do not remove any fields.
Do not include dialogue, monologue, lyrics, narration text, or script-only information inside the worksheet.

The purpose of this sheet is to visually communicate the scene structure to a video generation AI.
Prioritize visual blocking, camera work, character positions, background, lighting, movement direction, and emotional staging.

Fill the worksheet according to the following rules.

HEADER
- In the WORK TITLE field, write the title of the work or episode.
- In EP, write the episode number.
- In SCENE, write the scene number.
- In DURATION, write the total planned duration of the scene in seconds.

SCENE VISUAL (INITIAL STATE)
- Draw one large scene visual that represents the initial state before the cuts begin.
- This is the most important part of the worksheet.
- Show the stage/background clearly.
- Show the main characters’ starting positions, facing direction, distance, scale, and posture.
- Use simple arrows, labels, or motion guides only when they clarify staging.
- Do not draw a dramatic key visual unless it also explains the actual starting arrangement of the scene.
- The scene visual should function as a blocking map plus cinematic establishing image.

SCENE SUMMARY
- Write a concise four-line visual summary.
- Do not write dialogue.
- Describe only information useful for visual generation:
  1. scene mood / tension
  2. emotional direction
  3. lighting / color / atmosphere
  4. story progression or visual objective
- Keep each line short and concrete.

CUT LIST
- Fill up to 8 cut cells.
- Use only the required number of cuts.
- If fewer than 8 cuts are needed, leave the remaining cells blank.
- Do not enlarge unused cells.
- Each cut cell must contain:
  - a large cut number from 1 to 8
  - TIME as start–end seconds within the scene
  - CAMERA
  - CUT IMAGE
  - CUT SUMMARY

CUT NUMBER
- Keep the large number in the reversed-color badge.
- Use only the number, not “CUT 01” or similar text.

TIME
- Write the exact time range within the scene, for example:
  00:00–00:04
  00:04–00:08
- The cut times should connect continuously and match the total scene duration.

CAMERA
- Describe the camera in short English terms.
- Use clear production words such as:
  WIDE, MEDIUM, CLOSE, EXTREME CLOSE, LOW ANGLE, HIGH ANGLE, POV, OVER THE SHOULDER, TRACKING, DOLLY IN, PAN, TILT, HANDHELD, STATIC.
- Keep the camera field compact.

CUT IMAGE
- Draw the visual content of the cut.
- Show the main subject, framing, direction of movement, and important spatial relationship.
- Use arrows or short labels inside the image only when they help explain motion.
- Make the image readable at worksheet scale.
- Avoid excessive detail.
- Do not include speech balloons.
- Do not include written dialogue.

CUT SUMMARY
- Describe the visual action and direction of the cut in concise English.
- Focus on what moves, how the camera moves, what changes on screen, and what the viewer should feel.
- Do not write dialogue.
- Do not write literary prose.
- Do not include information that cannot be represented visually.

STYLE
- Keep the worksheet clean, readable, and production-ready.
- Use a rough storyboard drawing style, not a finished illustration style.
- Use grayscale or restrained color unless color is necessary for meaning.
- Preserve all template borders, headings, spacing, and the 4×2 cut grid.
- The final result should look like a filled professional storyboard worksheet for AI video generation.

Now fill the worksheet using the following scene information:

[WORK TITLE]
ここに作品名を書く

[EP]
ここに話数を書く

[SCENE]
ここにシーン番号を書く

[DURATION]
ここにシーン全体の秒数を書く

[SCENE CONTENT]
ここにシーン概要を書く

[CHARACTERS]
ここに登場人物、外見、衣装、初期位置を書く

[LOCATION / BACKGROUND]
ここに舞台背景を書く

[MOOD / LIGHTING]
ここにテンション、感情、ライティング、色調を書く

[CUT PLAN]
ここにカットごとの内容を書く。
各カットは以下の形式で書く：

CUT1.
 TIME:
 CAMERA:
 VISUAL ACTION:

CUT2.
 TIME:
 CAMERA:
 VISUAL ACTION:

CUT3.
 TIME:
 CAMERA:
 VISUAL ACTION:
```


## 出力ルール

- 絵コンテ画像は対象エピソードの `02_Anime/storyboards/<話数>/` に出力する。
- 1ページ目は `01_cut_sheet_01.png` のように、話数・用途・ページ番号が分かるファイル名にする。
- 生成AIで作成したビジュアルを使用する場合も、最終ページではテンプレートの罫線、列構成、CUT番号、本文の可読性を優先する。
