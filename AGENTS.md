# AGENTS.md

このプロジェクトでファイルを作成・整理するときは、README.md のディレクトリ構成に従う。

## 配置ルール

- プロジェクト全体で共有する世界観、歴史背景、基本ルールは `common/` に置く。
- 個別の物語に関するファイルは `stories/[Story_Title]/` に物語ごとに分けて置く。
- 物語ごとの制作物は、用途に応じて以下のカテゴリへ配置する。

## 物語フォルダ内の構成

- `01_Novel/settings/`: その物語固有の国、街、組織、特殊ルールなどの設定。
- `01_Novel/manuscripts/`: 小説本文、原稿データ。
- `02_Anime/designs/`: キャラクター、舞台、デザイン情報、画像生成プロンプト。
- `02_Anime/references/`: 動画生成用の設定画、三面図、表情差分、背景画などの参照素材。
- `02_Anime/storyboards/`: テキストコンテ、カットごとの画像・動画生成プロンプト。
- `03_Music/concepts/`: テーマソングやBGMのコンセプト、曲調、モチーフ。
- `03_Music/prompts/`: 音楽生成AI向けの歌詞、ジャンル設定、生成プロンプト。
- `04_Game/[Game_Title]/`: ゲームタイトルごとの制作物。

## 判断基準

- 複数の物語で共有する設定は `common/`、特定の物語だけに関係するものは `stories/[Story_Title]/` に置く。
- 新しい物語を追加する場合は、`stories/` 配下に物語名のフォルダを作成し、必要な制作カテゴリをその中に作る。
- 既存のカテゴリに当てはまるファイルは、独自フォルダを増やす前に該当カテゴリへ収める。
