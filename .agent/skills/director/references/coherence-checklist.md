# Internal Coherence Checklist

**For AI use only.** Run these checks silently before presenting the treatment to the user. Fix any failures before output. Do NOT show this checklist or its results to the user.

## When to Run

| Mode | When | Scope |
|------|------|-------|
| C (Original) | After writing treatment, before Confirm ① | Full checklist |
| D (Adaptation) | After condensing scenes, before Confirm ① | Full checklist |
| E (Montage) | After beat sheet, before Confirm ① | Quick checks only |
| A (Quick) | Before writing prompt | Quick checks only |
| B (E-commerce) | Before writing prompt | Quick checks only |

---

## Full Checklist (Modes C, D)

### 1. Causal Chain

For every scene transition, verify it's THEREFORE or BUT — never AND THEN.

- [ ] Every transition has a clear causal link
- [ ] At least 30% are BUT (complications)
- [ ] No scene could be removed without breaking the chain

**If failed**: Add a consequence or complication between disconnected scenes.

### 2. Character Consistency

- [ ] Every character action has explicit motivation
- [ ] No character contradicts their established personality without setup
- [ ] Characters REACT before they ACT (respond to what just happened)
- [ ] Emotional transitions are gradual (calm→curious→shocked, NOT calm→enraged)

**If failed**: Add a setup beat or change the action to fit the character.

### 3. Emotional Arc

- [ ] No two adjacent segments target the same viewer emotion
- [ ] There is a clear emotional peak (climax)
- [ ] The final emotion feels earned by what came before
- [ ] At least one moment of emotional contrast (high→low or low→high)

**If failed**: Insert a contrast beat — a brief reversal, surprise, or shift.

### 4. Information Flow

- [ ] Every scene gives the viewer at least one new piece of information
- [ ] The viewer is never confused about WHERE they are or WHO is on screen
- [ ] Key plot points are SHOWN (visually), not just TOLD (dialogue exposition)

**If failed**: Add visual cues or a brief establishing moment.

### 5. Dialogue

- [ ] Each line sounds like a real person speaking (not ad copy, not narration)
- [ ] No line repeats information the viewer already knows
- [ ] Silent beats exist where silence serves the story
- [ ] Every line has a clear purpose (reveal / character / conflict / emotion / subtext)

**If failed**: Cut purposeless lines. Rewrite exposition as natural speech.

### 6. Visual-Narrative Alignment

- [ ] Camera movement has narrative motivation (push-in = intimacy, pull-back = isolation)
- [ ] Lighting tracks the emotional arc (warm at hope, cold at despair)
- [ ] No dramatic camera work in calm scenes (technique shouldn't compete with content)

**If failed**: Simplify the technique or find a story reason for it.

---

## Quick Checks (Modes A, B, E)

### Q1: Is there a shift?
- Start state ≠ End state. If they're the same, add a transformation.

### Q2: Does every dialogue line serve a purpose?
- If any line is just filler, cut or replace it.

### Q3: Does the ending land?
- The final emotion should feel earned by what came before.

---

## Recovery Actions Summary

| Failure | Fix |
|---------|-----|
| AND THEN link | Add consequence or complication |
| Unmotivated action | Add trigger/setup earlier |
| Same emotion × 2 | Insert contrast beat |
| Viewer confusion | Add visual/dialogue cue |
| Purposeless dialogue | Cut it or give it a function |
| Gratuitous camera work | Simplify or justify narratively |
