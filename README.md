# AI動画生成テンプレート

本プロジェクトはAI創作活動のテンプレートセットです。


## プロジェクト構造

プロジェクト全体のデータおよび各物語における「小説」「映像」「音楽」の各アセットは、以下のようなディレクトリ構成で管理されます。

```text
/
├── README.md                   # 本ファイル（プロジェクト概要とディレクトリ構成）
├── common/                     # プロジェクト共通の世界観・設定を管理（全体で共有）
│   ├── 絵コンテルール.md        # 絵コンテ画像生成ルールファイル
│   ├── 絵コンテテンプレート.png  # 絵コンテ画像テンプレート
│   └── 動画生成ルール.md        # 絵コンテをもとに動画生成するためのルール
└── stories/                 # 各物語を管理するディレクトリ
    ├── [Story_Title]/       # 物語ごとのフォルダ（タイトル名等で作成）
    │   ├── 01_Novel/        # １．小説執筆
    │   │   ├── settings/    # 物語独自の世界観の設定
    │   │   └── manuscripts/ # 執筆原稿
    │   ├── 02_Video/        # ２．映像制作
    │   │   ├── designs/     # キャラクターデザイン、舞台デザイン、それらのプロンプト
    │   │   ├── references/  # キャラクターシート、設定画（動画生成用リファレンス素材）収録フォルダフォルダ
    │   │   └── storyboards/ # コンテ（テキスト）、画像/動画生成プロンプト、絵コンテ収録フォルダ
    │   └── 03_Music/        # ３．作曲
    │       ├── concepts/    # 楽曲のコンセプト
    │       └── prompts/     # 生成プロンプト
    └── [Another_Story]/     # 次の物語フォルダ
        └── ...
```

## 各ディレクトリの役割

### `common/` (共通設定)
プロジェクト全体（全物語）に共通する世界観、歴史背景、基本ルールなどの設定を配置します。各物語のベースとなる重要な基礎情報です。

### `stories/[Story_Title]/` (物語ごとのアセット群)
共通設定を広げる形で展開される「個別の物語」ごとにフォルダを作成します。この中で、３つの制作カテゴリに分けてファイル管理を行います。

#### `01_Novel/` (１．小説執筆)
- **`settings/`**: その物語にのみ登場する国、街、組織、あるいは特殊なルールなど、物語特有の設定ドキュメントを格納します。
- **`manuscripts/`**: 執筆した脚本の本文テキスト（原稿データ）を格納します。

#### `02_Video/` (２．映像制作)
- **`designs/`**: 当該物語に登場するキャラクターや舞台のデザインに関する情報、および画像生成用のプロンプトを管理します。
- **`references/`**: 動画生成AIに画像を読み込ませるためのリファレンス素材（三面図や表情差分、背景画などの設定画）を管理します。
- **`storyboards/`**: 映像の流れを指示するテキストコンテや、カットごとの画像・動画生成プロンプトを管理します。

#### `03_Music/` (３．作曲)
- **`concepts/`**: 物語のテーマソングやBGMなど、制作する楽曲のコンセプト、曲調、モチーフなどをまとめたテキストを格納します。
- **`prompts/`**: SunoやUdioなどの音楽生成AIに入力するための生成プロンプト（歌詞、ジャンル設定などを含む）を管理します。



## 生成ワークフロー

本テンプレートでは、脚本をもとにした演出、コンテづくり、映像制作をエージェンティックに行うことを想定しています。

### コンテ作り
「絵コンテ画像生成ルール」に示された、動画生成AI一回のリクエストで生成できる長さ（～15秒）を一枚のシートとして作成します。

脚本
↓
テキストコンテ（脚本を元に作成）
↓
絵コンテ（テキストコンテと絵コンテテンプレートを元に作成）
↓
動画生成（絵コンテとテキストコンテと、参照素材を元に作成）
